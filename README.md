# Setup

1. place menuboard folder in home/code or home/pi (I'm not entirely sure right now)
2. open terminal in menuboard folder and type "bash setup.sh"

# Shut down (READ BEFORE SETUP)

1. press alt+f4

# Update

1. shut down menuboard (alt+f4)
2. add/remove/edit files in menuboard/pages folder
3. open terminal in menuboard folder and type "bash run.sh"
